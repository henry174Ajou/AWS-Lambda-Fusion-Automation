Interface FunctionChainSpecification

This is a root object of a function chain specification JSON/YAML file. start here to write your function chain specification.

A function chain specification contains two information:

  1. source function information
  2. workflow structure

chain

chain: ChainItem[]
  • Defined in FunctionChainSpecification.ts:25

Ordered list of step.

The step means an acton that workflow will run. It will be a function, a conditional branch, iteration, or a message transformation.

This is a ordered list. The order of step is meaningful. The chain can be nested if the workflow contain fan-outs or conditional branches.
